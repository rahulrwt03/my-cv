# Resume Project

## Description
This project is a simple resume webpage created using only HTML. The purpose of this project is to practice and demonstrate basic HTML skills.

## Project Structure
- `index.html`: The main HTML file containing the resume content.

## How to Use
1. Clone or download the project to your local machine.
2. Open `index.html` in your web browser to view the resume.

## Features
- Basic HTML structure
- Sections for personal information, education, work experience, and skills
